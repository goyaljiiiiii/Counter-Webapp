# Counter App

## Overview
Counter App is a lightweight web application that lets users increment, decrement, and reset a counter. It features a clean UI, smooth animations, and responsive layout across devices.

## Features
- Increment, decrement, and reset the counter
- Responsive layout for mobile and desktop
- Lightweight and fast

## Tech Stack
- HTML
- CSS
- JavaScript

```
.

    └── assets
        ├── css

        └── js
            └── index.js
```

## Getting Started
1. Clone the repository:
```sh
git clone https://github.com/yourusername/counter-app.git
```
1. Open `index.html` in your browser.

## Usage
1. Click `+` to increase the counter.
1. Click `-` to decrease the counter.
1. Click `Reset` to reset the counter.
